## Project Overview

**Current Goal**: Build a code reasoning model (1-3B parameters) as the "thinking engine" for OpenCode agent.

The model should:
1. **Reason** - Analyze problems before solving
2. **Query RAG** - Actively call `<tool_call>search_rag(query)</tool_call>` when needed
3. **Generate Code** - Output code based on reasoning and retrieved knowledge

**Architecture**: TR-mamba2attn (based on arXiv:2602.12078)
**Deployment**: vllm with OpenAI-compatible API

## Common Commands


## Architecture

### Components (in `model.py`)

- **RMSNorm**: Root Mean Square Layer Normalization (Llama/Mamba style)
- **RotaryEmbedding**: Rotary Position Embedding (RoPE), supports seq_len up to 2048
- **MambaSSD**: State Space Duality block implementing Mamba-2's SSD algorithm
- **MambaHybridBlock**: Current implementation: Mamba → Mamba → Light Attention → SwiGLU MLP
- **Mamba2Block**: (To be implemented) Upgraded SSD block for TR-mamba2attn
- **Mamba2Attention**: (To be implemented) SSM-based attention for QKV processing

### Target Architecture: TR-mamba2attn

```
Mamba2Block → Mamba2Block → Mamba2Attention → MLP
```

**Target Parameters (1-3B)**:
| Parameter | Target Value |
|-----------|--------------|
| d_model | 512-768 |
| n_heads | 8-12 |
| d_state | 128 |
| head_dim | 64 |
| expand | 2 |

### Full Model (in `extreme_reasoning_model.py`)

- **ExtremeReasoningModel**: Complete causal language model
  - Token + Position embeddings with shared weights
  - Configurable layers, attention heads, and state dimension
  - Shared embedding/head weights (~30% parameter savings)
  - Built-in greedy generation

## Notes

- Training data: JSONL format with question + reasoning + tool_calls + tool_results + answer
- Generated by LLM (synthetic data) via generate_training_data.py
- The codebase uses no external dependencies beyond PyTorch
- Model includes parameter count printing via `_print_param_count()`
- Supports causal masking for autoregressive generation
- RoPE currently cached for max_seq_len=2048 (can be extended to 128K)
- Hardware: CMP 90HX (10G VRAM) / RTX 3080 for training
