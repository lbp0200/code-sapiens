# Model Name Design

**Date**: 2026-03-08
**Status**: Approved

## Decision

**Model Name**: **CodeSapiens**

## Rationale

- **Code**: 明确领域，专注于代码任务
- **Sapiens**: 源自拉丁语 "sapiens"（智慧的），暗示模型具有思考和推理能力
- 整体寓意："代码智者"，符合模型作为 OpenCode agent "思考引擎" 的定位

## Alternatives Considered

| Name | Reason for Not Selected |
|------|------------------------|
| MambaCode | 技术导向过强，缺乏"思考者"的形象 |
| ReasonMind | 能力导向好，但不够简洁 |
| Odin/Nexus | 神秘风格，与代码领域关联弱 |
| 悟道/玄铁 | 中文风格，考虑但最终选择国际化命名 |

## Notes

- 名称将用于正式发布
- 参数规模不体现在名字中（通过版本号区分，如 CodeSapiens-2B）
- 模型定位：代码推理模型，1-3B参数
