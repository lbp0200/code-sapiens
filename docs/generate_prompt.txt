# 生成训练数据的Prompt

你是一个数据生成器，生成用于训练"代码推理模型"的示例数据。

## 任务描述

生成JSON格式的训练数据，每条数据包含：
1. question: 用户的一个编程问题
2. reasoning: 模型解决这个问题时的思考过程
3. tool_calls: 模型主动查询RAG的动作（0-2个为宜，不要太多）
4. tool_results: 模拟RAG返回的结果（可以是代码片段、文档片段）
5. answer: 最终生成的代码

## 重要规则

1. **reasoning要详细**：展示完整的思考过程
2. **tool_calls要自然**：只有真正需要查资料时才调用RAG
3. **tool_results要真实**：模拟RAG可能返回的内容
4. **answer要正确**：最终代码要能正确运行

## 示例

输入问题类型：
- "写一个排序算法"
- "如何用Python读取CSV文件"
- "帮我写一个装饰器"
- "解释一下什么是闭包"
- "写一个二叉树遍历"

## 输出格式

每行一个JSON对象，不要其他内容：

{"question":"...","reasoning":"...","tool_calls":[...],"tool_results":[...],"answer":"..."}
{"question":"...","reasoning":"...","tool_calls":[...],"tool_results":[...],"answer":"..."}
